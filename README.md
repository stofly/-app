# 毛概答题app
这是我期末一个人敲的大作业，同时也为了能让同届同学考好毛概期末考试，加快了开发速度，中间存在很多简陋
# 主要技术
全部都是用最基础的activity和Fragment写的，用了okgo的网络框架其他貌似没啥的了
